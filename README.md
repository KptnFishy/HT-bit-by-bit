# Ski-Tracker-App

Dieses Projekt enthält eine Ski-Tracker-App, die im Format eines 10h-Hackathons entwickelt wurde.

## Features
- Crash-Erkennung mit automatischen Notrufen
- Fahrt-Tracking
- Statistiken
- Stat-Leaderboards
- Routen-Sharing mit Map-Integration

## Tech-Stack
### Frontend
- Expo
- React Native
- NativeWind

### Backend
- Supabase

## Setup

1. Installation Dependencies:
```bash
cd frontend
npm install
```

2. App starten:
```bash
npx expo start
```
